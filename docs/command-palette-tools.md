# command-palette-tools

`command-palette-tools` は、コマンドパレットを素早く開くためのショートカット導線を追加するプラグインです。

## 使い方

1. Plugin Manager で `command-palette-tools` を有効化します。
2. 上部ツールバーの **Cmd** ボタンをクリックします。
3. もしくはプラグインコマンドから **コマンドパレットを開く** を実行します。

## 追加される機能

- Toolbar: `open-command-palette`（ラベル: Cmd）
- Command: `open-command-palette`
