# rapid-layer-tools

`rapid-layer-tools` は、Rapid AI レイヤーをワンクリックで切り替えるためのプラグインです。

## 使い方

1. Plugin Manager で `rapid-layer-tools` を有効化します。
2. 上部ツールバーの **AI** ボタンをクリックすると、Rapid レイヤーの表示が切り替わります。
3. もしくはプラグインコマンドから **Rapid AIレイヤー切替** を実行します。

## 追加される機能

- Toolbar: `toggle-rapid-layer`（ラベル: AI）
- Command: `toggle-rapid-layer`
